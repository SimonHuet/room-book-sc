# Room-book-sc

> Technical test from Sharing cloud to handle room booking


![Continuous Integration](https://github.com/SimonHuet/room-book-sc/actions/workflows/ci.yml/badge.svg)
## Getting started

### Prerequisites

- Docker and docker compose  

Or
- node
- `npm install`

### Start the project

With Docker: 
```bash
docker compose up
```

Without: 
```
npm start
```
